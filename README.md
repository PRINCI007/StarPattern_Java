# ⭐ Java Star Pattern Programs

This repository contains a collection of **Java programs** that demonstrate various **star (`*`) pattern printing** techniques. 
These programs are commonly used in beginner-level coding exercises and help understand **nested loops**, **control flow**, and **basic logic building** in Java.

## 🚀 Features

- Multiple types of star patterns:
  - Right-angled triangle
  - Left-angled triangle
  - Pyramid
  - Inverted patterns
  - Diamond shapes
  - Hollow patterns
- Clean and readable code
- Suitable for beginners


## 🛠️ Requirements

- Java Development Kit (JDK) 8 or higher
- A code editor (like VS Code, IntelliJ IDEA, or Eclipse)
